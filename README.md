# React-JS--Crud-Mysql

## Installation

### Client

  1. `$ cd client`
  2. `$ npm install`
  3. `$ npm start` 
  
### Server

  1. mudar o nome do mySQL `server/index.js`
  2. `$ cd server`
  3. `$ npm install` 
  4. `$ npm rum devStart`
  
### Front-end

 + `@material-ui`
 +  `Axios`

### Back-end

+ `express`
+ `mysql`
+ `nodemon`
+ `cors`
